# 校园二手交易平台 - 开发环境配置与下载指南

既然我们确认了 **Trae + 微信开发者工具** 的开发模式，并且需要从头搭建环境，请按照以下步骤依次下载和安装所需软件。

---

## 1. 前端与小程序运行环境

### 1.1 Node.js (前端必备引擎)
Node.js 是前端运行和打包的基础环境。
*   **下载地址**：[https://nodejs.org/zh-cn/](https://nodejs.org/zh-cn/)
*   **下载版本**：选择 **"LTS (长期维护版)"**，通常是 v18.x 或 v20.x。下载 Windows 安装包 (`.msi`)。
*   **安装注意**：一路点击“下一步 (Next)”即可，不需要修改默认设置。安装程序会自动帮你配置环境变量。
*   **验证安装**：在 Trae 的终端 (Terminal) 中输入以下命令，如果能看到版本号说明成功：
    ```bash
    node -v
    npm -v
    ```

### 1.2 微信开发者工具 (小程序预览与调试)
用于实时查看我们在 Trae 中写的 Uni-app 小程序代码效果。
*   **下载地址**：[https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
*   **下载版本**：选择 **Windows 64位** 稳定版 (Stable Build)。
*   **安装注意**：正常安装即可。安装后打开软件，使用你的微信扫码登录。
*   **关键配置**：为了让 Trae 能自动把代码同步到微信开发者工具，你需要打开微信开发者工具，点击右上角齿轮 **设置 -> 安全设置**，将 **“服务端口”** 开启。

---

## 2. 后端 Java 运行环境

### 2.1 JDK 17 (Java 运行环境)
因为我们将使用较新的 Spring Boot 版本，建议安装 JDK 17。
*   **下载地址**：[https://adoptium.net/zh-CN/temurin/releases/](https://adoptium.net/zh-CN/temurin/releases/)
*   **下载选择**：操作系统选 `Windows`，架构选 `x64`，版本选 `17`。下载 `.msi` 格式的安装包。
*   **安装注意**：在安装向导的 "Custom Setup" 步骤中，请确保把 **"Set JAVA_HOME variable"** 选项从红叉改为 **"Will be installed on local hard drive"**（这非常重要，省去了你手动配置环境变量的麻烦）。
*   **验证安装**：在终端中输入：
    ```bash
    java -version
    javac -version
    ```

### 2.2 Maven (Java 依赖包管理)
*   **下载地址**：[https://maven.apache.org/download.cgi](https://maven.apache.org/download.cgi)
*   **下载选择**：下载 `Binary zip archive` (例如 `apache-maven-3.9.x-bin.zip`)。
*   **安装与配置**：
    1. 将压缩包解压到一个你专门存放软件的目录（比如 `D:\Environment\maven`）。
    2. 右键“此电脑” -> 属性 -> 高级系统设置 -> 环境变量。
    3. 在“系统变量”中点击“新建”，变量名填 `MAVEN_HOME`，变量值填你刚才解压的目录路径。
    4. 找到系统变量中的 `Path`，双击编辑，新建一行填入 `%MAVEN_HOME%\bin`。
*   **验证安装**：在终端中输入：
    ```bash
    mvn -v
    ```

## 3. 数据库与缓存环境

### 3.1 MySQL 8.0 (关系型数据库)
用于存储平台的核心数据。
*   **下载地址**：[https://dev.mysql.com/downloads/installer/](https://dev.mysql.com/downloads/installer/)
*   **下载选择**：选择较大的那个离线安装包（通常几百MB）。
*   **安装注意**：
    *   Setup Type (安装类型) 选择 **"Server only"** (仅安装服务端即可)。
    *   在 Authentication Method (认证方式) 步骤，选择 **"Use Legacy Authentication Method"** (传统认证，兼容性更好)。
    *   **⚠️ 极其重要**：在设置 Root 密码时，请务必设置一个你记得住的密码（例如：`123456` 或 `root`），后端代码连接数据库时需要用到这个密码！

### 3.2 Redis (内存缓存)
用于实现高并发和防止商品超卖。Redis 官方没有 Windows 版，但微软开源团队和社区提供了 Windows 版本。
*   **下载地址**：[https://github.com/tporadowski/redis/releases](https://github.com/tporadowski/redis/releases)
*   **下载选择**：下载最新的 `.zip` 压缩包（例如 `Redis-x64-5.0.14.1.zip`）。
*   **安装与运行**：
    1. 解压到指定目录。
    2. 进入解压后的目录，双击运行 `redis-server.exe`。此时会弹出一个黑黑的终端窗口，显示一个 Redis 的图形Logo，**只要这个黑窗口不关，Redis 就处于开启状态**。
    3. *（可选，方便测试）* 双击运行同目录下的 `redis-cli.exe`，如果能输入 `ping` 并返回 `PONG`，说明缓存环境已成功开启。
