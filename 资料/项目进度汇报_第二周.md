## 三、本周修复的关键 Bug

| Bug | 影响 | 修复方案 |
|-----|------|----------|
| `msg` vs `message` 字段不一致 | 移动端错误提示全部失效 | 统一改为 `msg` |
| 无 CORS 配置 | 所有前后端请求被浏览器拦截 | 新增 `CorsConfig` Bean |
| 无分页插件 | MyBatis-Plus 分页不生效 | 注册 `PaginationInnerInterceptor` |
| Token 从未校验 | 任何人可操作发布/下单/审核 | 新增 `TokenInterceptor` |
| 用户 ID 硬编码 | 所有订单/商品归同一个人 | 从 Token 中动态获取 userId |
| 前端无路由守卫 | 未登录可直接访问后台 | 添加 `router.beforeEach` |
